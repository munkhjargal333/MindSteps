# Сэтгэцийн эрүүл мэндийн систем

## 1. Төслийн танилцуулга

Энэхүү төсөл нь хэрэглэгч өөрийн сэтгэл зүйн байдлыг өдөр тутам тэмдэглэх, сэтгэл хөдлөлөө ажиглах, үнэт зүйлсээ тодорхойлох замаар өөрийгөө илүү сайн ойлгож, сэтгэцийн эрүүл мэнддээ анхаарах боломж олгох веб систем юм.

## 2. Багийн гишүүд

* **Ц. Мөнхжаргал** – Хөгжүүлэгч

## 3. Суулгах заавар

### Backend

```bash
cd backend
go mod tidy
# өгөгдлийн сангаас model үүсгэх
go run cmd/gen/main.go
# backend сервер ажиллуулах
go run cmd/api/main.go
# unit test ажиллуулах
go test ./test/unit/...
```

### Frontend

```bash
cd frontend
yarn install
yarn run dev
# integration test ажиллуулах
yarn test
```

## 4. Хийсэн зүйлс

### 4.1 Технологийн стек

* **Backend:** Golang (Fiber)
* **Frontend:** Next.js, Tailwind CSS
* **Өгөгдлийн сан:** Neon (PostgreSQL)
* **Deployment:** Render, Vercel

### 4.2 Реализацлагдсан гол функцууд

* Хэрэглэгчийн бүртгэл, нэвтрэх систем
* Өдрийн тэмдэглэл бүртгэл
* Үнэт зүйл бүртгэл
* Сэтгэл хөдлөлийн бүртгэл
* Зорилго үүсгэх, удирдах
* Бясалгал (demo хувилбар)

### 4.3 Аюулгүй байдал

* SQL Injection халдлагаас сэргийлсэн *(done)*
* XSS халдлагаас хамгаалалт *(not yet)*
* Нууц үгийг bcrypt ашиглан хэшлэх *(done)*
* JWT токен дээр суурилсан баталгаажуулалт *(done)*

## 5. Тулгарсан бэрхшээлүүд

Хөгжүүлэлтийн явцад техникийн болон багийн ажиллагааны олон бэрхшээл тулгарсан ч эдгээр нь цаашдын төслүүдэд илүү төлөвлөгөөтэй, чанартай ажиллах чухал туршлага болсон. ## Хамгаалалтын илтгэлийн бүрэн текст 

### Нээлтийн үг 

Сайн байцгаана уу, багш нар болон сонсогчид оо.
Өнөөдөр та бүхэндээ миний боловсруулсан **“Сэтгэцийн эрүүл мэндийн систем”** төслийг танилцуулж байна.

Сүүлийн жилүүдэд сэтгэлзүйн стресс, сэтгэл хөдлөлийн тэнцвэр алдагдах асуудал ихсэж байгаа ч хүмүүс өөрийн сэтгэл зүйгээ тогтмол тэмдэглэж, ажиглахад зориулсан энгийн, ойлгомжтой систем хомс байна. Үүнийг шийдэх зорилгоор уг төслийг хэрэгжүүлсэн.

Төслийн гол зорилго нь хэрэглэгч өөрийн **сэтгэл хөдлөл, өдрийн тэмдэглэл, үнэт зүйлсээ бүртгэх замаар өөрийгөө илүү ойлгож**, сэтгэцийн эрүүл мэнддээ анхаарах боломж олгох явдал юм.



### Системийн танилцуулга

#### 1. Хэрэглэгчийн бүртгэл

Системийг ашиглахын тулд хэрэглэгч эхлээд бүртгүүлнэ.

* Шинэ хэрэглэгч имэйл болон нууц үгээр бүртгүүлнэ
* Нууц үг нь bcrypt ашиглан hash хийгдэж хадгалагддаг
* Аюулгүй байдлыг хангах үүднээс JWT токен ашиглаж баталгаажуулалт хийсэн

Энэ нь хэрэглэгчийн хувийн мэдээллийг хамгаалах үндсэн механизм юм.

#### 2. Нэвтрэх, гарах

Бүртгэлтэй хэрэглэгч системд нэвтэрч:

* Энгийн хэрэглэгч
* Админ хэрэглэгч гэсэн эрхээр нэвтрэх боломжтой.

Session менежмент JWT дээр суурилсан бөгөөд token expiration ашиглан аюулгүй байдлыг хангасан.

#### 3. Үндсэн функцууд 

Системийн гол цөм нь дараах функцууд юм:

* **Өдрийн тэмдэглэл бичих** – хэрэглэгч тухайн өдрийн бодол, мэдрэмжээ тэмдэглэнэ
* **Сэтгэл хөдлөлийн бүртгэл** – сэтгэл хөдлөлийг сонгон бүртгэж, давтамжийг ажиглах боломжтой
* **Үнэт зүйл, зорилго үүсгэх** – өөрийн баримтлах үнэт зүйл, зорилгоо тодорхойлно
* **Бясалгал (demo)** – сэтгэл тайвшруулах зорилготой туршилтын функц

Эдгээр нь хэрэглэгчийг өөрийгөө ойлгох, өөрчлөлтөө анзаарахад чиглэсэн.

#### 4. Админ панел 

Админ хэрэглэгч:

* Хэрэглэгчдийн мэдээллийг хянах
* Системийн ерөнхий статистик харах
* Контентын зохицуулалт хийх
  боломжтой.
